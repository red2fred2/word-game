# Word Game
A boggle-like game run as a web app.

### Requirements
* Node
* Docker
* Docker Buildx

### Installing dependencies
```npm install -D```

### Building
1. ```gulp```
2. ```rolldown -c```

or run the build script
```build.sh```

### Running
1. ```docker compose up -d --build```

or run the run script
```run.sh```

### Development
1. ```rolldown -wc```
2. ```docker compose watch```
